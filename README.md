##About lorenz system

The lorenz system is a set of three differential equations that describe chaotic flow. it’s often used to study weather and dynamical systems. the equations are:

dx/dt = σ(y - x)  
dy/dt = x(ρ - z) - y  
dz/dt = xy - βz

common parameters:
σ = 10, ρ = 28, β = 8/3
